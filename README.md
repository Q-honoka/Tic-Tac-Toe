# [三目並べ] 

<img width="80%" height="80%" alt="image_game" src="https://github.com/user-attachments/assets/ce047878-ed62-4f62-a16a-dd911bf0f409" />

マウスで遊べるテーブルゲームです。ESCキーで終了します。
縦、横、斜めのいずれか１列に自分のコマを３つそろえたほうが勝ちです。勝敗がつかないままマスが埋まってしまった場合は引き分けになります。

## 🎮 ゲーム概要
- **ジャンル**: テーブルゲーム
- **プラットフォーム**: PC (Windows)
- **制作期間**: 16日間
- **言語**: C++
- **使用ツール**: VisualStudio2026, DxLib

## 🌟 実装のポイント
- **コマの設置処理の単純化**: マスのクリック時にコマを生成するのではなく、はじめに非表示で設置することでクリック時の処理を簡単にしました。
- **勝利判定のパターン化**:　勝利パターンが決まっているため二次元配列でそのパターンを保持して、勝利判定を簡単にしました。

## 💡 こだわった点 
- **コマの依存関係**: ゲーム進行を管理するクラスではなく、ボードのみがコマを持つようにして依存関係が複雑化しないようにしました。
